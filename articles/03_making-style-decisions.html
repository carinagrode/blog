Making Style Decisions

Sooo many decisions! It's really interesting, how much detail there is in one simple web page.

Which font looks nice, what's the best font size, space between paragraphs?

I make decisions and funny things happen: I remove the bullet points from the menu, but the bullet point in the footer stays there and makes itself comfy. I add space after paragraphs, but headings and images are no paragraphs, so they stay untouched.

Everything is quite obvious and makes sense. But only after I tried something. So much to keep in mind. You change one thing and have 5 more challenges.

Today I have:

  • Created the links for the navbar and the footer
  • Added some more styling to the text
  • Played around with space after paragraphs and before bullet points

Next Steps:

  • Create an About Page
  • Create a sidebar
